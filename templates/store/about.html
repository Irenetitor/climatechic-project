About ClimateChic

ClimateChic is a unique platform that combines fashion with weather insights. My goal is to help you choose the perfect outfit based on the current weather conditions in any city that you live in or are going to travel to. Whether it's a sunny day or a chilly evening, I've got you covered with the best fashion recommendations.


Developer - Irene Martin Campo

  • • I am a passionate web developer who is currently learning full stack development.
  • • With a keen interest in both technology and fashion, I decided to create ClimateChic to merge these two passions.
  • • Constantly experimenting with new technologies, I aim to provide a seamless and enjoyable user experience through this project.
  • • Through ClimateChic, I hope to help users make informed fashion choices while also honing my skills as a full stack developer.

Technologies Used

  • • Python & Django: The backend of the application is built using Django, a high-level Python web framework that encourages rapid development and clean, pragmatic design.
  • • JavaScript: Used for client-side scripting to enhance user interactions and validate forms.
  • • HTML & CSS: The structure and styling of the web pages are created using HTML and CSS, ensuring a responsive and visually appealing design.
  • • API: APIs are used to validate city names entered by users, get weather info for entered city and sample product lists ensuring accurate weather-based recommendations.
  • • Bootstrap: A front-end framework used to create responsive and mobile-first web pages.
